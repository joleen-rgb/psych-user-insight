三个关键差异

  1. 躯体化共现4%——她们的焦虑已经导致了身体症状（心慌、手抖、胸闷）
  2. 抑郁共现7.5%——部分人已经从"睡不好"滑向了情绪障碍

用户旅程

1. 触发期
某次重要事件前（考试/汇报/面试）失眠 → 或长期累积的职场/育儿压力 → 或更年期开始后激素变化导致睡眠障碍 → "怎么突然睡不着了"
2. 尝试期
试褪黑素、中药、呼吸法 → 无效 → 试"频率""白噪音" → 部分有效但不稳定
3. 焦虑叠加期
开始害怕"今晚又睡不着" → 对失眠本身的恐惧形成新焦虑层 → 一部分人出现躯体化症状（心悸、胸闷）→ 开始疑病 → "我是不是得了什么大病" → 更睡不着
4. 搜索爆发期
搜"胡思乱想怎么停""失眠怎么办""焦虑症需要吃药吗" → 疯狂收藏方法帖 → 收藏比2.49说明她们是真的在"囤方法"
5. 分叉
a.就医 → 被诊断焦虑症 → 纠结吃不吃药 → "焦虑症吃药是利大于弊吗"
b.自救 → 但碎片化方法不成体系 → 课程介入点
c.维持 → 每晚听频率/白噪音勉强入睡 → 治标不治本
