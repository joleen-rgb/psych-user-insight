课程产品全景图

第七章 -- 基于12个用户群体洞察的产品架构

01 获客层（免费/极低价）

产品对应入口形态
"你属于哪种类型"自测 自我标签收集者 小程序/H5测试 → 测完导向课程
"课程 vs 咨询"对比页 咨询价格受阻者 落地页——打价格和"给方法"的差异化
"我们知道你被坑过" 课程信任危机（负评是正评的2.3倍） 获客话术第一句直接回应信任问题

02 核心课程（按心理恶化程度分层）

层级对应人群核心内容数据验证
入门层：认知框架 高敏感 + 内耗 + 原生家庭 命名模式 + 生理解释 + 日常策略 命名帖收藏比 0.8-1.2
实操层：即时工具 睡前思维 + 情绪失控 睡前关机法 / 情绪急救包 / 场景化练习 "秒睡法"收藏比 2.49
导航层：就医教育 躯体化 + 就医彷徨 何时就医 / 精神科流程 / 药物预期 "不想留就诊记录" 1932评论

03 场景包（按生命阶段分线）

场景对应人群独有内容数据验证
妈妈线 全职妈妈 + 高三家长 + 休学家庭 不崩溃的育儿 / 代际传递阻断 / 每日微练习 / 休学应对 "每日抄写"收藏比 1.40
婚姻线 婚姻 + 原生家庭(婚姻重现) 在关系里不那么累 / "各过各的"策略 "看见需求"收藏比 1.12
更年期线 更年期 食疗方 / 激素科普 / 失眠管理 / 子女指南 食疗收藏比 2.70

05 延伸机会（新兴人群启示）

方向对应新人群机会描述
"从学员到倾听师"成长路径 倾听师转行者 课程同时满足自我成长和职业转型——学完可以"接单"帮别人
AI时代的关系能力课 AI依恋者 当AI能24小时陪伴，"在真人关系中获得安全感"反而成为需要学习的能力
休学家庭指南 休学家庭 同时面向家长和孩子——"休学不是终点，是孩子在说'我需要帮助'"

06 贯穿原则

30% 命名（获取信任）+ 70% 方法（创造价值）

每个模块的内容配比：用户愿意为"被说中"付出注意力，但愿意为"能照着做"付出金钱。

获客话术的起手式："我们知道你被坑过"

876条课程消费帖中负面评价是正面的2.3倍——信任危机是第一道墙。不是"我们的课程多好"，而是"我们知道你被坑过/试过很多都没用——这次不一样，因为XXX"。

课程的终极卖点不是"治愈你自己"，而是"为了你的孩子/为了你的生活，先理解你自己"——利他动机比自利动机更容易驱动行动。

16个人群不是16个市场，是同一条河流的16个截面。课程应该沿着河流来设计，而不是在每个截面上建一座独立的桥。
