依恋关系共现36%——是所有群体中仅次于婚姻群体（59%）的第二高。 说明大量用户做心理测试的真正动机不是"了解自己"，而是"理解为什么我的关系总是出问题"。

用户旅程

  • 1. 关系/情绪触发 → 某次冲突/分手/内耗 → "我为什么总是这样？"
  • 2. 标签搜索 → 搜"性格测试""心理测试" → 做大量测试 → 获得一堆标签："焦虑型依恋""INFJ""讨好型人格""高敏感"
  • 3. 标签收集期 → 不断做新测试 → 每个结果都觉得"是我！" → 收集越来越多标签
  • 4. 解释成瘾 → 用标签解释所有行为："我之所以这样是因为我是回避型" → 标签变成了固化的自我定义
  • 5. 两个分叉：
    a. 从标签走向行动 → "知道了我是XX型，然后呢？" → 课程介入点
    b. 停留在标签层 → 继续测试 → 永远在"了解自己"但从不改变

    子群7："我是什么"集体交换 ▼

    评论区的核心互动模式是：

    • 用户A："我选了3"
    • 帖主："选3的你是XXX型"
    • 用户B："也是3！太准了！"
    • 用户C："我是2和4之间犹豫"
    评论区变成了一个"身份交换市场"——每个人都在寻找自己的位置。 这种互动模式是课程社群可以复制的——"你是哪种类型？在社群里找到同类"。

    互动行为规律

    这个群体的互动模式和其他所有群体完全不同：

    内容类型赞数评论数收藏说明
    直觉测试帖极低（200-800）极高（5000-32000）极低参与>消费
    人格命名帖极高（2万-12万）中（1000-4000）高消费>参与
    依恋解读帖高（5000-30000）高（500-1600）高两者兼有
    测试帖是全数据集中"评论/赞比"最极端的内容形态——39.5（TA对你什么心思）、42.5（最先想到的字）、17.9（焦虑值）。 没有任何其他内容形态能产生如此密集的评论。

    对课程的核心启示

    1. 引流层必须包含测试——这是被数据验证的最强互动机制
    2. 测试结果应该导向课程模块："你是焦虑型依恋→推荐学习路径A"
    3. 测试的质量可以远超现有的"凭直觉选数字"——提供更有深度的自测工具本身就是课程的价值之一
