痛苦时刻——发帖时间

21:00是唯一的尖峰（34帖）——深夜暴食高发时段。 其次是17:00-19:00（63帖），对应下班后/放学后。凌晨1:00有6帖，对应深夜暴食后的自责。

关键发现：这个群体的"抑郁"标签共现率很低（1.5%），但帖子里的情绪强度极高——"我好痛苦""救救我""我该咋办"。 说明她们不认为自己"抑郁"，而是把痛苦归因到"吃"和"身材"上。这是课程切入的重要角度——帮她们看到"不是你管不住嘴，是你的情绪需要出口"。

用户旅程

  • 1. 身材不满触发 → 某次照镜子/称体重/被评价 → 开始节食/减肥
  • 2. 控制期 → 严格热量控制 → 短期见效 → "我可以的"
  • 3. 反弹崩溃 → 某次情绪波动触发暴食 → 一次吃5000大卡 → 极度羞耻
  • 4. 节食-暴食循环 → 节食→暴食→自责→更严格的节食→更猛烈的暴食 → 循环无法打破
  • 5. 寻找解释 → 搜"为什么控制不住" → 遇到"暴食不是馋，是感受太少了" → 第一次把暴食和情绪联系起来
  • 6. 系统求助 → 但找不到"心理+饮食"的整合方案 → 课程介入点
