用户旅程

  1. 1. 命名痛苦
    搜"为什么我总想太多" → 遇到"高敏感的人在学校根本活不下去" → "是我！" → 学到标签
  2. 2. 标签认同
    搜"高敏感人格" → 大量消费描述性内容 → 开始自称"本人高敏感"
  3. 3. 天赋or诅咒的纠结
    想相信是天赋，但日常体验确实是痛苦 → 这个阶段停留最长
  4. 4. 寻找出路
    搜"高敏感怎么办" → 收藏大量方法帖，但碎片化不构成可执行系统
  5. 5. 系统性解决
    大部分人停在第4步 -- 这是课程介入的点
